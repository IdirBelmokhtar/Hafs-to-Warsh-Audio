avant tous, you must put your data in the folder named 'All' we have [114 surah, 6236 ayah].

To convert this audio from hafs to warsh you need to execute this files one by one:
1- run_change_ayah_name_like_warsh_from_hafs.bat
2- run_check_and_merge_differances.bat
3- run_check_and_cut_differances.bat to see files was not converting
4- cut the data showed in your commande line manualy
	(you can execute cut_audio.html to help you in cuting) but you need to instal "ffmgep" and start traitement.php from php server file
5- run rename_folders_and_files.py

NOW everthing is DONE.

you can share the folder All in (Google Drive in my case).
6*- go to google drive/ my program to make JSON file
	- get the links from drive with google sheet help (Apps Script)
	- copy them in (all links.xlsx)
	- before execute get_and_stor_drive.py to get the output.json (Final file), renamme variable 'name' according to the name of Qarii insead the py file.
7*- rename output.json as you need and enjoy.
